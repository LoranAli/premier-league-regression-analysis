# När datan talar – Vad avgör en säsong i Premier League?
*En multipel regressionsanalys av faktorer bakom sportslig framgång*


## Projektbeskrivning

Det här projektet undersöker vilka faktorer som bäst förklarar ett lags poäng i Premier League under en säsong. Analysen bygger på statistik från säsongen 2024/2025 och genomförs med hjälp av multipel linjär regressionsanalys.

Målet är att förstå om faktorer som offensiva och defensiva prestationer, tekniska samt fysiska attribut kan användas för att förklara sportslig framgång, och om det finns vissa variabler som är extra betydelsefulla.

## Metod och data

- **Datakälla:** [FBref.com](https://fbref.com/en/comps/9/stats/Premier-League-Stats) – statistik för samtliga lag i Premier League 2024/2025.
- **Analyserade variabler:** Skott på mål (SoT), expected goals (xG), passningsprecision (Cmp), lyckade tacklingar (TklW), defensiva misstag (Err), total distans (TotDist), och andra utvalda nyckeltal.
- **Statistisk metod:** Multipel linjär regression med urval av variabler via AIC och multikolinjäritetstest (VIF).
